Excel Formulas and Function References

Use this page to reference how to set up functions and formulas.

Excel Operators

  • Comma , Separates more then one cell. Such as (A1, A16). Excel will use only A1 and A16.
  • Colon: Creates a range of cells. Excel will look at all the data between the two nameed cells. Such as A1:A16, Excel will use every cell and data from rows 1-16 in the A column.

Functions: Functions are built-in algorithms that perform complex caluculatios in Excel.

A function has two parts: the name and an argument.


  • Function Name: The name indicated the type of math Excel will perform such as SUM for addition or AVERAGE to calucate the average number across a range of data.
  • Argument: The argument is the value that a function uses to perform operations. This includes numbers, text, cell references, and names.

A function name gives Excel the instruction of what to do. The arguments are the TOOLS to perform the function. Think of it like playing a sport. You might play Tennis (Function) and you play with a Racket, Court, Net, and Tennis Ball (Arguments).

Here are few examples of functions using the different operators:

Example 1: =SUM(10,20,40) Adds the numbers 10 + 20 + 40
Example 2: =SUM(A1:A13) Adds the cells A1 through A13
Example 3: =SUM(A1, B13, C19) Adds only the cells A1 + B13 + C19
Example 4: =SUM(A1:C19) Adds all of the cells A1 through C19.

Creating 3D References

In Excel, you can refere to the same ell or range of cells on muliple sheets. This is called a 3-D reference. The function will not just look at a singe worksheet, but pull the data from mulitple worksheets into one function or formula.

Let's look at an example using the formula =SUM(SALES:COMMISION!B3)

    Steps to build a 3-D reference

  1. Click on the cell where you want to enter the function
  2. Type = (equal sign), enter the name of the function. In the above example we use SUM.
  3. Type an opening paranthesis.
  4. Click the tabe for the first worksheet you want to reference.
  5. Hold down SHIFT and click the tabe for the last worksheet you want to reference.
  6. Select the cell or range of cells that you want to reference.
  7. Complet ehe formula and then press ENTER.

VLOOKUP

A VLOOKUP is a vertical lookup. VLOOKUP is used when you need to perform calculations that reference a table with a range of values.

Let's look at how a VLOOKUP works first. A VLOOKUP searches down the first column of a table to find the value you want.

For example, lets say we are looking for all the sales people named "Peter Pan". VLOOKUP will look vertically in the column until it finds all the matches for "Peter Pan". Next it will look across a specified number of columns and return a value in the original cell selected. This is most commonly used when we need to cross reference incomes or sales with a range.


VLOOKUP Function Builder has three required parts

  1. Lookup_value: This is the Value you want to find. Use a cell reference here.
  2. Table Array: The entire table you want to look for the matching value.
  3. Col_index_num: Which Column number (left to right) should Excel vertically look through.
