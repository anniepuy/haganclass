Lesson 4: Editing Cell Data

Objective: Students will be able to edit data in a cell.

By now you are familar with how to simply edit the data in a cell by deleting the contents and typing in side the cell.

Your mouse cursor will change to indicate or show different types of editing modes. Here are a few styles:

Handle Name Pointer Uses
Normal Select a cell or range of cells.
Fill Handle Copy cell contents to the next or adjacent cell
I-Beam Edit the contents inside an active cell or formula bar
Row Resize Resizes the height of a row.
Column Resize Resizes the width of a column.

It can be tricky to get the handle on working with the different cursors. As you practice with Excel, take your time with getting the right cursor for the proper action.

Whenever you type inside a cell, the mirror image of the type will also show in the formula bar.
