Lesson 22: IF OR Functions

Objective: Students will practie evaluting multiple cell references against a true or false statement with and IF OR function.

Remember, IF functions evaluate a value against a true or false statement. But what if you need to evaluate a value against more then one statement? That is where adding the OR comes in to play.

=IF(OR(B3>70, C3>70, D3>70), "Old", "Youngster")
