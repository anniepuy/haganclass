Lesson 6: Managing Worksheet Views and Formating worksheets

Objective: Students will be able to use different methods of navigation to view, edit, and manipulate data in a spreadsheet.

This lesson will have you practice changing the worksheet views using the View tab. Most importantly, you will use the Page Layout View.

The Page Layout View allows you to edit the header and footer on individual worksheets. This cannot be done in normal view.

In Excel, there are THREEparts to each header and footer.

  • Left Header/Footer
  • Center Header/Footer
  • Right Header/Footer

Each part can be customized with text, dates, file names, and so much more!
