Lesson 3: Understanding Formulas

Objective: Students will be able to apply the basic formulas of SUM and AVERAGE, to a selected cell range.

Excel is powerful because it allows for quick and accurage calculations of numbers. A formula is an equation in a worksheet.

Excel formulas are set up like the right side of an equation. They will always start with the "=" sign.

Here are some examples:

Operator Purpose Example
+ Addition =B9+B24
- Subraction =C10-C44
/ Division =B9/B24
* Multiplication =B9*B24

The above formulas work great when you are only using two cells. If you want to reference more than two cells, then Excel has some functions that make it easier.

These are found under the "Formulas" tab and "Insert Function" option. All of these functions start with an = sign, then a word, and the cell reference in parenthesis.

Function Purpose Example
SUM Addition of multiple cells =SUM(B4:B67)
MAX Displays the Maximum value of a given set of cells. =MAX(D2:E43)
