Lesson 19: Using CountIF Functions to Analyze Data

Objective: will apply knowledge of "IF" functions and apply it to the functions of COUNTIF.

IF Functions have three main parts that are required:

  • Logical Test: A reference cell is tested against an inequality such as equal to, less than, or greater than.
  • Value if True: If the referenced cell MATCHES the test, what will Excel return.
  • Value if False: If the referenced cell DOES NOT MATCH the test, what will Excel return.

The COUNTIF, AVERAGEIF, and SUMIF functions are speficif IF functions that use the same three parts as a regular IF. The difference is that COUNTIF will only count the cell entries that match the test. SUMIF will only add the cell values that match true to the logical test, and AVERAGEIF will only average cell data that match the logical test.
