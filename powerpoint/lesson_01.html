Lesson 2: Creating Presentations from Outline

Objective: Students will be able to create a new PowerPoint presentation using an outline in Word(.doc) or from a text file (.txt). They will apply this knowledge to creating their own outline from their MineCraft Farm planning sheet.

Today you will learn how to import a Word document and a text file into PowerPoint. Just like Excel and Word, importing a different type of file requires the use of the ribbon bar.

For this lesson, you will learn to use the Insert Slide -> From Outline on the Home Tab of the ribbon bar.
